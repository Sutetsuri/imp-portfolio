1. JQuery questions:

What you can do with the jQuery?

Well, jQuery is a JS library that is meant to make client-side scripting into a more pleasant experience. It follows the principle "do more, write less", making code-heavy tasks less bulky. Taking into account how popular it is, you can do pretty much anything with it.

How it can help you as web developer?

It simplifies DOM manipulation and AJAX calls, which are pretty complicated on their own. However, it can do a lot more then that. Too much to cover, in fact.

If you think our earlier participant list app, in what kind of issues you could use jQuery?

Well, you can add some cool features to it, like editing the list on the go.

What are CDNs? What kind of benefits you get if you use them?

CDN is short for Content Delivery Network and can also somethimes be referred to as mirroring. CDN is a system of servers aimed at delivering web content to the user based on the geographical location. When user is trying to access the service, CDN server redirects the request to the nearest CDN server to the user.
