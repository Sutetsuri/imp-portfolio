1. Defenition list:

HTML - (HyperText Markup Language) a markup language for creation of Web pages

CSS - (Cascading Style Sheets) language that dictates how HTML elements are displayed

Javascript - a very versatile programming language, for the most part is used to make HTML and CSS code interactive

HTTP - an application layer protocol used globally in World Wide Web

HTML5 - the newest version of the HTML

Dynamic HTML (DHTML) - broad term for pages that are dynamic after the page is fully loaded made possible with client-side scripting language such as JS
